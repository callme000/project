# TaskFlow Suite

A comprehensive, responsive Task Management Suite built with HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

## Features
- **Responsive Dashboard**: Manage tasks across To Do, In Progress, and Done columns.
- **LocalStorage Persistence**: Your tasks stay saved in your browser.
- **Multi-page Architecture**: Includes Home, About, Contact, Task Archive, Team, and FAQ sections.
- **Priority Tracking**: Color-coded task priorities.
- **Form Validation**: Robust client-side validation for task entry and contact forms.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Bootstrap 5
- **Icons**: Font Awesome (Planned)
- **Data**: Browser LocalStorage

## Project Structure
- `index.html`: Home page.
- `dashboard.html`: Main application interface.
- `assets/css/`: Custom styles.
- `assets/js/`: Application logic and storage modules.
